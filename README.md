# Hotel Maria: Customer Segmentation Analysis 

## Project Overview
This project involves a comprehensive **segmentation analysis** for **Hotel Maria**, a charming four-star hotel located in the heart of Lisbon, Portugal. Using a dataset covering three years (2015-2018), the goal is to identify distinct customer segments to help management develop personalised offerings and improve marketing efficiency.

---

## Dataset Description
The dataset provides a "treasure trove" of information regarding customer behaviour, preferences, demographics, and geographical data. Key variables include:

* **Demographics:** Age and Nationality.
* **Revenue Tracking:** Total spent on lodging (room, crib) and "Other Revenue" (food, beverage, spa).
* **Booking Behaviour:** Average lead time, cancellations, no-shows, and total checked-in bookings.
* **Stay Details:** Total person/nights and room/nights, as well as the number of days since the first and last stay.
* **Customer Preferences:** Specific requests for floor levels (High/Low/Medium), bed types (King/Twin), and amenities like quiet rooms or no alcohol in the mini-bar

---

## Methodology & Goals
The analysis follows a professional marketing analytics framework:

1.  **Exploratory Data Analysis (EDA):** Identifying patterns and descriptive statistics to gain initial insights.
2.  **Segmentation Strategy:** Applying clustering techniques to the entire customer base or specific subgroups.
3.  **Optimisation:** Evaluating the optimum number of clusters using elbow or silhouette plots, while exploring alternative techniques for computational efficiency given the large dataset size.
4.  **Actionable Insights:** Creating visuals to present cluster characteristics and discussing the results within the context of the current hotel industry landscape.

---

## Technical Implementation
* **Tools:** R Markdown was used to create a professional marketing report and ensure reproducible logic.
* **Anonymisation:** Data privacy was maintained using SHA2-256 hash strings for customer names and document IDs.
* **Visuals:** A visually engaging slide deck was created to summarise key insights for management.
